# TodoList

## Features
-- Add new tasks  
-- Delete tasks  
-- Press **Enter** to quickly add a task  
-- Dark Mode / Light Mode toggle  
-- Tasks persist after page refresh using LocalStorage  
-- Responsive event handling   using event delegation
